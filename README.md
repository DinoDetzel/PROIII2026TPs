## Trabajo Práctico 1 - TechNova Solutions

**Objetivo:**  
Desarrollar un sitio web estático utilizando HTML y CSS, aplicando estructura semántica, navegación entre páginas y formularios.

**Descripción:**  
Crear un sitio web para una empresa o proyecto tecnológico (tema libre), compuesto por múltiples páginas conectadas.

**Páginas requeridas:**
- `index.html` → Home con descripción e imagen principal  
- `servicios.html` → Mínimo 3 servicios (imagen, título, descripción)  
- `equipo.html` → Mínimo 3 integrantes (nombre, rol, imagen)  
- `pedido.html` → Formulario completo de solicitud de servicios  
- `contacto.html` → Formulario de contacto  
- `faq.html` → Mínimo 5 preguntas frecuentes  

**Requisitos técnicos:**
- Uso de etiquetas semánticas (`header`, `nav`, `main`, `footer`)  
- Navegación entre páginas  
- CSS externo  
- Uso de box model (`margin`, `padding`, `border`)  
- Clases e IDs  
- Mínimo 3 imágenes  
- Rutas relativas  
- Estructura ordenada de carpetas  

**Trabajo en equipo:**
- Uso de Git y GitHub  
- Ramas: `main`, `dev` y una por alumno  
- Cada integrante debe hacer commits y al menos un merge con Pull Request  

**Entrega:**
- Repositorio en GitHub  
- Se evalúa el último merge en `main`  
- Incluir documentación y datos del grupo en el README
